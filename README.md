# ballsdex-packs

**A Ballsdex extension package that adds a pack collection and opening system.** Users can claim daily and weekly packs via `/pack daily` and `/pack weekly` commands.

## Installation

Add the following into your `config/extra.toml` file.

```toml
[[ballsdex.packages]]
location = "git+https://github.com/seiftheman/ballsdex-packs.git#main"
path = "packs"
enabled = true
```
